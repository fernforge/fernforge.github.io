Alchemy

Alchemy makes AI write prose a person would actually write.

Coding agents write competent text and then give themselves away: the empty "not just X, it's Y" contrast, a pile of words like robust and seamless, an em-dash in every other sentence, a closing paragraph that recaps what you just read. Alchemy is a ruleset your agent reads before it writes. It names those patterns and tells the model to cut them, so the output reads like it came from someone who knows the topic.

It's about a thousand tokens of rules, not a service or an API. Point your agent at it and the writing changes.

Install

  • Node / any coding agentnpx @fernforge/alchemy init
  • Pythonpip install alchemy-writing
  • MCP server (serves the ruleset to any agent)npx @fernforge/alchemy-mcp

What it does to a sentence

Before

Redis isn't just a database — it's a powerful, robust caching
layer that unlocks the full potential of your entire stack.

After

Redis is an in-memory store. Put your cache and sessions in
it and your database stops being the bottleneck.

Same claim, minus the tells: no "isn't just X, it's Y", no stacked adjectives, no em-dash, no "full potential". The rewrite says the one concrete thing instead.

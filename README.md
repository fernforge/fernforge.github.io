# fernforge.github.io

Source for the fernforge org site at https://fernforge.github.io/.

A small shop building developer tools: [Alchemy](https://github.com/fernforge/alchemy)
(a ruleset that makes AI write like a person), [mcp-conform](https://github.com/fernforge/mcp-conform)
(an MCP spec linter), and [devnotes](https://fernforge.github.io/devnotes/).

The site is one static `index.html`, served by GitHub Pages from the default branch.
